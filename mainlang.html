Основы языка PascalABC.NET

На этой странице собраны основные конструкции, операторы и примеры программ на языке PascalABC.NET

Содержание

  1. Структура программы
  2. Вывод на экран
  3. Целочисленные переменные
  4. Считывание (ввод с клавиатуры) переменных

Структура программы

Основная программа в PascalABC.NET размещается между операторами begin и end.

Begin
// Основная программа
end.

Переменные объявляются до основной программы:

var название1, название2, название3: тип1;
var название4, название5, название6: тип2;
Begin
// Основная программа
end.

Комментарии

В PascalABC.NET комментарии могут быть сделаны с помощью // или { }

info

Оператры внутри программы разделяются точкой с запятой:

Begin
    оператор1;
    оператор2;
    оператор3;
    оператор4;
end.

Вывод на экран

Для вывод информации на экран в PascalABC.NET применяются операторы write и writeln.

Оператор writeln, в отличии от write, осуществляет после вывода переход на новую строку. Например:

program outputExample; { название программы }
Begin
    write('Hello ');
    write('world!');
    { На экран будет выведено "Hello world!" в одну (!) строку }
    writeln('Hello ');
    write('world!');
    { На первой строке будет выведено "Hello ", а на второй "world!" }
end.

Если нужно вывести несколько элементов на экран, то можно указывать их через запятую. Вывод значения переменной на экран:

program exampleOutput;
var height := 5;
Begin
    write('Высота: ', height, ' метров'); { выведет на экран: 'Высота: 5 метров' }
end.

Ограничено только памятью компьютера

В PascalABC.NET, чтобы присвоить значение переменной нужно воспользоваться оператором :=. Пример: height := 5; Объявление и использование целочисленных переменных:

program integerExample;
var a, b, c: integer;
Begin
    a := 3;
    b := 5;
    c := a + b;
    write(c); { Выведет 8 }
end.

Считывание (ввод с клавиатуры) переменных

Чтобы считать переменную с клавиатуры в PascalABC.NET применяется операторы read и readln. Последний после считывания осуществляет переход на новую строку. Пример использования оператора read с целыочисленным типом integer:

program integerExample;
var a, b, c: integer;
Begin
    write('Введите первое число: ');
    readln(a);                                          { Вводим 7 }
    write('Введите второе число: ');
    readln(b);                                          { Вводим 8 }
    write('Сумма чисел ', a, ' и ', b, ' равна ', a+b); { Будет выведено: Сумма чисел 7 и 8 равна 15 }
end.
